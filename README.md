# 📚 Book Store Management System

Application for managing a bookstore Api , 
built using Node.js, Express, and MongoDB.
This project allows users to browse books and administrators to manage the inventory.

## 🚀 Features
* User Authentication: Secure login and registration using JWT or Sessions.
* Book Management: CRUD operations (Create, Read, Update, Delete) for books.
* Dynamic Templates: Server-side rendering using EJS.
* Database: NoSQL storage with MongoDB and Mongoose schemas.
* Responsive UI: Clean and simple design for all devices.

## 🛠️ Tech Stack
* Backend: Node.js, Express.js
* Frontend: EJS (Embedded JavaScript), CSS
* Database: MongoDB
* Tools: Mongoose, Dotenv, Git/GitHub
